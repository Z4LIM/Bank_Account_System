# 自动化任务

## Cron 任务

| 名称 | 调度 | 功能 | 状态 |
|------|------|------|------|
| github-daily | 每天 8:00 | GitHub 每日简报 | error |
| earnings-tracker | 每周日 18:00 | 财报追踪 → 飞书 | idle |

## 财报追踪

- 追踪公司：NVDA, MSFT, GOOGL, META, AMZN, TSLA, AMD
- 每周日 18:00 自动搜索财报日历
- 整理发送到飞书私聊
